Unit Converter

Convert between different units instantly. Type in any field to see conversions in all units.

How to Use

Type a value in any unit field, and all other units will automatically convert to the equivalent value. No need to select a "from" or "to" unit - just type and see instant results.

Example: Type "100" in the mm field, and you'll immediately see "10" in cm, "0.1" in m, "3.94" in inches, etc.
